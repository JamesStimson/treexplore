Achievements

The analysis presented here demonstrates that it is possible to build and implement a rigorous framework to compare disparate transmission tree inference methods. It gives the community a basis from which it is possible to evaluate new and evolving techniques in a way that is statistically analysable.

The results obtained give confidence in the methods employed: each method's results form a coherent cluster, and the relationship between the clusters plausibly reflects the differences of the underlying models.

Issues

The methods compared are implemented in many different technologies, using their own bespoke formats for most of their inputs and outputs. A challenge has been to understand and interpret these differences enough in order to start building a pipeline to integrate the essential results.

Because SCOTTI allows hosts to be re-infected (as opposed to only being able to be infected once) it sometimes generates cycles in the inferred transmission network. This invalidates the assumptions made in the tree distance model [6], and for the purpose of this analysis, these cases are excluded.

Anomalies can occur in the larger timed transmission trees for SCOTTI due to the necessity of applying a simple rule to estimate transmission times.

Further Work

There are several ways in which the work of this project can be developed:

  • Extension to more methods. Fourteen methods are identified in [6], including the four analysed here, and there are certainly more available. The challenges here are (a) the investment needed to run them in a consistent manner and (b) to be able to extract comparable information from their outputs.
  • Investigating the effect of modifying the priors. These can have a large impact on the results and associated likelihoods, and can be modified in many different ways.
  • Statistical analysis of the posterior uncertainty.
  • Further use of visualisation tools. For example, transmission network edges can be weighted according to likelihood.
  • Building a robust pipeline to compare the methods.

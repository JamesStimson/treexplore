Individual Plots

For each point in the MDS plot, there are three detailed representations of data from that sample:

  • A transmission network (see Medians page for more details)
  • A timed transmission tree (see Medians page for more details)
  • A timed phylogenetic tree. In the case of TransPhylo, this is fixed and supplied as an input. For the other three methods, this is inferred alongside the transmission tree by the MCMC process.
These three plots are shown beneath the main MDS plot.

It is important to note that the phylogenetic trees inform the transmission trees, but are not equivalent. The labels on the tips (always on the right hand side) show the same host identifiers as with the transmission trees. However, the branch points in the trees represent branches in the family tree of the host pathogen, whereas the branches in the transmission tree represent transmission events - the jumping of the pathogen from one host to another.

Depending on the method used, different plotting capabilities are available for the phylogenetic trees, so the following is shown:

  • For Beastlier and SCOTTI, a simple tree with the dates indicated at the branch tip labels
  • For PhyBreak, branches change colour to indicate when transmission between hosts occurs, the three colours being re-used for various hosts
  • For TransPhylo, each host is individually coloured and transmission events are indicated with red asterisks
